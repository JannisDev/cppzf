Programmstruktur

Das Programm läuft zeilenweise von oben nach unten ab:


#include <iostream>

int main() {
    // Code ...
    return 0;
}

Die erste Zeile inkludiert eine Headerdatei welche für die ein und ausgabe im Programm wichtig ist. Sie wird daher immer benötigt.

Die Funktion main() enthält alle Anweisungen für das Programm.
Da sie vom Typ Integer ist, muss sie auch einen zurückgeben. Daher steht am ende der Funktion return 0;.


Kommentare

Kommentare werden verwendet, um Abschnitte des Codes auszuschließen.

Du wirst auch in diesem Dokument einige Komentare finden.


Einzeilige Kommentare

Einzeilige Kommentare werden verwendet um einzelne zeilen zu kommentieren:

// Dies ist ein Kommentar

Dabei muss der Kommentar nicht immer am Zeilenanfang beginnen:

return 0; // Dieser Kommentar steht nach einer Anweisung

Mehrzeilige Kommentare

Mehrzeilige Kommentare werden verwendet mehrere zeilen in einem Stück zu kommentieren:


/*
Dies ist ein Kommentar
Und er hat mehrere Zeilen
*/

Operatoren

Operatoren werden verwendet, um Datentypen zu verknüpfen oder zu verändern.

Folgende sind die wichtigsten Operatoren:


=       // Zuweisungsoperator
+       // Addition
-       // Subtraktion
*       // Multiplikation
/       // Division
%       // Modulo (Rest bei Division)
++      // Inkrement (Erhöhung um 1)
--      // Dekrement (Verringerung um 1)

Variablen

Variablen werden verwendet, um verschiedenste Daten die zur ausführung des Programms benötigt werden zu speichern.

Folgende sind die wichtigsten Variablen:


int     // Ganzzahlvariable
float   // Fließkomma-Variable mit einfacher Genauigkeit
double  // Fließkomma-Variable mit doppelter Genauigkeit
char    // Zeichenvariable
bool    // Boolesche Variable
struct  // Struktur mit benutzerdefinierten Feldern

Deklaration/ Initialisierung

Deklaration (Variable anlegen)

Bevor eine Variable genutzt werden kann, muss sie Deklariert werden.
In folgendem Beispiel wird eine variable mit dem Datentyp int und dem Namen myVariable deklariert:

int myVariable;

Initialisierung (Wert Setzen)

Die Initialisierung kann erst stattfinden, nachdem die Variable Deklariert wurde.
In folgendem Beispiel wird dem zuvor erstellten Integer mit dem Namen myVariable den wert 43 gegeben:

myVariable = 43;

Sollte dem Datentyp char einen Wert zugewiesen werden, muss dieser in Hochkommata geschrieben werden:

myChar = "a";

Bei Flieskommazahlen ist zu beachten dass die Zahl mit einem Punkt statt Komma geschrieben wird:

myDouble = 6.9;

Kombination

Die Deklaration und Initialisierung kann auch in einem Schritt durchgeführt werden.
In folgendem Beispiel wird eine variable mit dem Datentyp float und dem Namen myVariable deklariert und gleichzeitig initialisiert:

float myVariable = 6.9;


Verwendung

Nachdem die Variable erfolgreich deklariert und initialisiert wurde, kann Sie im Code einfach durch Ihren namen aufgerufen werden.

In folgendem Beispiel wird in der main() Funktion eine Variable des Datentyps int mit dem Namen rueckgabewert und
dem Wert 0 erzeugt und am ende der Funktion zurückgegeben:


int main() {
    int rueckgabewert = 0;

    return rueckgabewert;
}

Bedingungen und Logik

Um in einem Programm Logik zu verwenden gibt es verschiedene möglichkeiten.


If-/ Else-Blöcke

If

Ein If-Block wird dazu verwendet um eine Anweisung oder eine Gruppe von Anweisungen nur dann auszuführen wenn eine Bedingung erfüllt ist.
Dabei kann beispielsweise eine Variable auf Ihren inhalt überprüft werden.


if (myInteger == 420) {
    // Code wird ausgeführt wenn myInteger den Wert 420 hat.
}

In einem If-Block können folgende Vergleichsoperatoren verwendet werden:


==  // Wert a ist exakt Wert b                  Bsp. (myVariable == 420)
!=  // Wert a ist nicht Wert b                  Bsp. (myVariable != 420)
<   // Wert a kleiner als Wert b                Bsp. (myVariable < 420)
<=  // Wert a kleiner als oder exakt Wert b     Bsp. (myVariable <= 420)
>   // Wert a größer als Wert b                 Bsp. (myVariable > 420)
>=  // Wert a größer als oder exakt Wert b      Bsp. (myVariable <= 420)

Außerdem können mehrere vergleiche kombiniert werden.


%%  // Vergleich a und b müssen 'true' sein     Bsp. (myVariable == 420 %% myChar == "a")
||  // Vergleich a oder b müssen 'true' sein    Bsp. (myVariable == 420 || myVariable == 69)

Else

Ein Else-Block folgt immer unmittelbar nach einem If-Block und wird dann ausgeführt wenn die bedingung des If-Blocks nicht zutrifft.


if (myVariable == 420) {
    // Code wird ausgeführt wenn myVariable den Wert 420 hat.
} else {
    // Code wird ausgeführt wenn myVariable nicht den Wert 420 hat.
}

Else If

Mit Else If können mehrere Abfragen hintereinander gekettet werden.
Somit kann eine weitere bedingung abgefragt werden, sollte die erste nicht zutreffen.


if (myVariable == 420) {
    // Code wird ausgeführt wenn myVariable den Wert 420 hat.
} else if (myVariable == 69) {
    // Code wird ausgeführt wenn myVariable den Wert 69 hat.
} else {
    // Code wird ausgeführt wenn myVariable nicht den Wert 420 oder 69 hat.
}

Switch

Mit einem Switch können mehrere bedingungen gleichzeitig abgefragt werden.
Beispiel eines Switch:


switch (myVariable){
case 420:
    // Code wird ausgeführt wenn myVariable den Wert 420 hat.
    break;
case 69:
    // Code wird ausgeführt wenn myVariable den Wert 69 hat.
    break;
default:
    // Code wird ausgeführt wenn myVariable nicht den Wert 420 oder 69 hat.
    break;

Schleifen

Schleifen werden verwendet, um eine Anweisung oder eine Gruppe von Anweisungen wiederholt auszuführen, solange eine Bedingung erfüllt ist.

While-Schleife

Die while-Schleife führt eine Anweisung oder eine Gruppe von Anweisungen aus, solange die Bedingung wahr ist.
Dabei wird die Bedingung genau gleich gestaltet wie bei einem If-Block.


while (myBool) {
    // Code ...
}

Do-While-Schleife

Die do-while-Schleife führt eine Anweisung oder eine Gruppe von Anweisungen mindestens einmal aus, und wiederholt sie solange die Bedingung wahr ist.


do {
    // Code ...
} while (myBool);

For-Schleife

Die for-Schleife wird verwendet, um eine Anweisung oder eine Gruppe von Anweisungen eine bestimmte Anzahl von Malen auszuführen.


for (int i = 0; i <= 4; i++) {
    // Code wird in diesem fall 5 mal ausgeführt.
}

Arrays

Ein Array ist eine Sammlung von Elementen desselben Datentyps, die durch einen gemeinsamen Namen und einen Index identifiziert werden.

Arrays in C++ werden deklariert, indem der Datentyp und die Größe des Arrays angegeben werden. Die Größe des Arrays gibt an, wie viele Elemente es enthält.

Arrays werden mit eckigen Klammern ([ ]) indiziert. Der Index beginnt bei 0 und endet bei der Größe des Arrays minus eins.

Beispielcode:


// Deklaration und Initialisierung eines Arrays
Datentyp arrayName[Größe] = {Element1, Element2, ..., ElementN};

// Zugriff auf ein Array-Element
arrayName[Index] = Wert;


Pointer

Ein Pointer ist eine Variable, die die Adresse eines anderen Objekts im Speicher speichert.

Pointer in C++ werden deklariert, indem der Datentyp des Objekts angegeben wird, auf das der Pointer verweisen soll, gefolgt von einem Stern (*) und dem Namen des Pointers.

Um den Wert, auf den der Pointer zeigt, zu erhalten, verwenden Sie den Dereferenzierungsoperator (*) vor dem Pointer-Namen.

Beispielcode:


// Deklaration eines Pointers
Datentyp* pointerName;

// Initialisierung eines Pointers
Datentyp* pointerName = &VariableName;

// Zugriff auf den Wert, auf den der Pointer zeigt
*pointerName = Wert;
